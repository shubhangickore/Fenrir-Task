# Project Overview

- This project is a frontend implementation for the Fenrir Security Design Challenge. It demonstrates my skills in React.js, React Router, and Tailwind CSS. 
- The application features clean UI/UX, navigation between pages, and interactive components.

## Features
Responsive design using Tailwind CSS

Routing using React Router

Theme toggle (light/dark mode) implementation

Interactive components for smooth user experience

Clean and modular component structure

## Live Demo

- Live URL: [https://fenrirtask1.vercel.app/]

## Tech Stack
- React.js – Frontend library for building UI

- Tailwind CSS – Styling and responsive design

- React Router DOM – Client-side routing

- Vite – Project bundler
